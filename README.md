# AI-Crypto-Market-Monitor-Automation-Workflow-with-Make.com-
AI Crypto Market Monitor


## 🚀 AI Crypto Market Monitor

An automated cryptocurrency monitoring system that tracks real-time market data, detects significant price movements, and sends instant alerts using workflow automation.

---

## 📸 Project Overview

This project uses **automation + API integration** to monitor cryptocurrency markets in real time and notify users when major price changes occur.

It also includes a **no-code frontend dashboard** for visualization.

---

## ⚙️ Tech Stack

* **Make.com** – Workflow automation
* **CoinGecko API** – Crypto market data
* **Telegram Bot** – Real-time alerts
* **Google Sheets** – Data logging
* **Lovable** – Frontend dashboard

---

## 🧠 How It Works

```
Webhook → API Request → Data Processing → Filtering → Alerts → Logging → Dashboard
```






### Step-by-step:

1. Webhook triggers the workflow
2. Fetch crypto data from API
3. Process each coin using iterator
4. Apply filtering logic (±5% price change)
5. Send alerts via Telegram
6. Store results in Google Sheets
7. Display data on dashboard

## ⚡ Challenges & Solutions

**Problem:** API returns bulk data
**Solution:** Used iterator to process each item

**Problem:** Needed dual-condition filtering
**Solution:** Implemented (>5% OR < -5%) logic

**Problem:** No-code UI integration
**Solution:** Connected Lovable dashboard via webhook
